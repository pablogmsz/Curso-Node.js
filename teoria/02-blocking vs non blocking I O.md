# Blocking
- Las instrucciones son bloqueantes, por lo tanto la ejecución del programa es mucho más lenta.
- Debo evitar este tipo de instrucciones porque las respuestas se ven perjudicadas.

# Non Blocking I/O
- Las instrucciones no son bloqueantes.
- Como no son bloqueantes, se realizan de manera simultánea.
